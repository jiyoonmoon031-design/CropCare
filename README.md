# 🌾 농장지킴이 (Farm Guardian)

**AI 기반 작물 병해 진단 및 농장 모니터링 모바일 플랫폼**

농장지킴이는 작물 이미지를 기반으로 병해를 진단하고, 농장 및 구역 단위의 병해 이력을 관리하며, 대시보드를 통해 병해 발생 추세를 분석할 수 있도록 돕는 AI 기반 모바일 애플리케이션입니다.

---

# 📖 Description

농장지킴이는 **컴퓨터 비전과 AI 기술을 활용하여 작물 이미지를 분석하고 병해를 자동으로 진단하는 모바일 플랫폼**입니다.

사용자는 작물 이미지를 촬영하거나 업로드하여 병해 여부를 확인할 수 있으며,  
진단 결과는 **농장 및 구역 단위로 기록되어 대시보드를 통해 병해 발생 추세를 분석**할 수 있습니다.

또한 **Grad-CAM 기반 시각화**를 통해 AI가 어떤 영역을 보고 판단했는지 설명하며,  
질병 정보와 방제 방법을 제공하여 **비전문 사용자도 쉽게 농작물 건강 상태를 관리할 수 있도록 지원합니다.**

---

# 🚀 주요 기능

## 🔍 1. 병해 진단 (Diagnosis)

- 🌿 식물 이미지 업로드 또는 카메라 촬영
- 📷 AI 추론 전 이미지 품질 검사
- 🧠 병해 분류 (Disease Classification)
- 📍 병변 위치 탐지 (Object Detection)
- ⚠️ 심각도 평가 (Severity Estimation)
- 📊 모델 신뢰도 표시 (Confidence Score)
- 🔥 Grad-CAM 기반 시각화
- 💊 방제 추천 (Treatment Recommendation)
- 📚 질병 상세 설명 페이지

---

## 🗂 2. 이력 관리 (History Management)

- 📝 진단 기록 저장
- 🔎 날짜 / 작물 / 질병 / 구역 기준 검색 및 필터
- 📍 농장 / 구역 단위 관리
- ✔️ 조치 여부 관리  
  - 미조치  
  - 조치완료

---

## 📊 3. 분석 대시보드 (Analytics Dashboard)

- 📊 질병 발생 빈도 분석
- 🌾 작물별 질병 분석
- 🗺 구역별 병해 지도
- 🚨 위험 구역 강조 표시
- 🔔 경고 알림
